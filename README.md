# Packages & fonts


```
pacman -S --needed - < bare-metal-pkgs.txt 
(for physical hardware)                

pacman -S --needed - < vm-pkgs.txt 
(for virtual machine)

pacman -Qqe > pkgs.txt 
(creates backup of current packages)

cp -r .config /home/"Your user name"
(copies configs to home directory)
```

# **Custom keybindings**

```F5``` - screenshot

```Alt+w``` - terminal

```Alt+c``` - close working window

```Alt+ctrl+f``` - firefox

```Alt+ctrl+t``` - telegram

```Alt+ctrl+n``` - nemo file manager

```F1``` - volume down

```F2``` - volume up

```F3``` - mute
